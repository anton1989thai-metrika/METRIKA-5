# Шаг 1: Что такое сервер для почты?

## Что нужно для собственного почтового сервера?

### Вариант 1: VPS (Виртуальный приватный сервер) ⭐ Рекомендуется

**VPS** - это виртуальный сервер в интернете, который вы арендуете у хостинг-провайдера.

**Примеры провайдеров:**
- **DigitalOcean** (от $4/месяц) - https://www.digitalocean.com
- **Linode** (от $5/месяц) - https://www.linode.com
- **Vultr** (от $2.50/месяц) - https://www.vultr.com
- **Hetzner** (от €4/месяц) - https://www.hetzner.com
- **Timeweb** (от 200₽/месяц) - https://timeweb.com
- **Selectel** (от 300₽/месяц) - https://selectel.ru

**Что нужно:**
- Минимум 1 GB RAM
- 20 GB диска
- Ubuntu 20.04 или 22.04 (рекомендуется)
- Статический IP адрес (обычно включен)

**Плюсы:**
- ✅ Полный контроль
- ✅ Низкая стоимость
- ✅ Можно установить что угодно
- ✅ Собственный IP адрес

**Минусы:**
- ❌ Нужно настраивать самому
- ❌ Нужны базовые знания Linux

---

### Вариант 2: У вас уже есть сервер

Если у вас уже есть:
- VPS сервер
- Выделенный сервер
- Сервер в дата-центре
- Домашний сервер с белым IP

**Тогда можно использовать его!**

**Важно:** 
- Сервер должен иметь **белый (публичный) IP адрес**
- Порт 25 должен быть открыт (для приема почты)
- Рекомендуется Ubuntu/Debian

---

### Вариант 3: Облачный сервер (AWS, Google Cloud, Azure)

Если у вас есть аккаунт в:
- Amazon AWS
- Google Cloud Platform
- Microsoft Azure
- Yandex Cloud

**Можно использовать их!**

**Важно:** 
- Многие провайдеры блокируют порт 25 по умолчанию
- Нужно запросить разблокировку порта 25
- Обычно дороже, чем VPS

---

## Какой вариант выбрать?

### Для начала (рекомендуется):
**VPS от DigitalOcean или Timeweb**
- Простая регистрация
- Низкая стоимость
- Хорошая документация
- Поддержка на русском (Timeweb)

### Если уже есть сервер:
**Используйте его!** Просто скажите мне:
- IP адрес
- Какая ОС установлена
- Есть ли root/sudo доступ

---

## Что делать дальше?

### Если у вас НЕТ сервера:

1. **Зарегистрируйтесь у провайдера** (например, Timeweb или DigitalOcean)
2. **Создайте VPS:**
   - Выберите Ubuntu 22.04
   - Минимум 1 GB RAM, 20 GB диска
   - Выберите регион (ближе к России)
3. **Получите:**
   - IP адрес сервера
   - Логин и пароль для SSH
4. **Сообщите мне** - я помогу настроить!

### Если у вас УЖЕ ЕСТЬ сервер:

**Сообщите мне:**
1. IP адрес сервера: `_____________`
2. Какая ОС: `_____________` (Ubuntu/Debian/CentOS)
3. Есть ли SSH доступ: `да/нет`
4. Есть ли root/sudo доступ: `да/нет`

И я помогу настроить почтовый сервер на нем!

## Вопросы для уточнения:

❓ **У вас уже есть сервер?**
- Да → сообщите IP и ОС
- Нет → нужно зарегистрироваться у провайдера

❓ **Какой провайдер предпочитаете?**
- Российский (Timeweb, Selectel) - проще оплата
- Зарубежный (DigitalOcean, Vultr) - обычно дешевле

❓ **Какой бюджет?**
- Минимум: ~200-300₽/месяц (российские провайдеры)
- Или $4-5/месяц (зарубежные)
