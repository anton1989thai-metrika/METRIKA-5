# Быстрая настройка получения писем

## Шаг 1: Настройте IMAP в .env

Откройте файл `.env` и добавьте/обновите настройки:

```env
IMAP_HOST="imap.yandex.ru"
IMAP_PORT="993"
IMAP_USER="reg@metrika.direct"
IMAP_PASS="ваш_пароль_от_яндекс_почты"
```

## Шаг 2: Включите IMAP в Яндекс.Почте

1. Зайдите на https://mail.yandex.ru
2. Настройки → Почтовые программы
3. Включите "С сервера imap.yandex.ru по протоколу IMAP"
4. Если требуется, создайте пароль для приложения

## Шаг 3: Синхронизируйте письма

### Вариант A: Через интерфейс
1. Откройте http://localhost:3000/email
2. Нажмите кнопку "Синхронизировать" в правом верхнем углу
3. Письма появятся в папке "Входящие"

### Вариант B: Через API
Откройте в браузере или через curl:
```bash
curl -X POST http://localhost:3000/api/emails/sync
```

## Шаг 4: Автоматическая синхронизация (опционально)

Для автоматического получения писем каждые 5 минут, добавьте в cron:

```bash
# Откройте crontab
crontab -e

# Добавьте строку (замените на ваш URL)
*/5 * * * * curl -X POST http://localhost:3000/api/emails/sync
```

Или используйте системный таймер (systemd) на Linux сервере.

## Готово! 

Теперь письма, приходящие на `reg@metrika.direct`, будут синхронизироваться с вашим сайтом.

**Важно:** Письма все еще будут приходить в Яндекс.Почту, но также будут копироваться на ваш сайт через IMAP.

Для полной независимости от Яндекс нужно настроить собственный почтовый сервер (см. EMAIL_SYNC_SETUP.md).
